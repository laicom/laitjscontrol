    关于"javascript控件"一词我还没见过有人这么用过，实际上它不是真正意义上的"控件"，而是一个独立的
javascript函数，与一般的javascript程序不同的是它是基于DOM（文档对象模型）和面向对象的方法设计，拥有属性、
方法和事件以及特定的界面元素。
